#### SSH key 생성
키를 생성하면 공개키와 비공개키가 만들어진다. 
SSH Key를 통해서 서버에 접속 할 때 Unix 계열(리눅스, 맥)에서는 ssh-keygen 사용


이 중에 비공개키는 client 에 위치해야 하고, 공개키는 server(remote)에 위치해야 한다.
